
**変更点**
- スロットの外観
- scoreの表示
- アニメーションの追加
- リールの画像表示

**操作**
1. startボタンをクリック　
1. stopボタンが三つあり、それぞれをクリック　
1. 結果がモーダル表示される、ブラウザ上のどこかをクリックするとモーダル表示が介助される。
1. 結果はscoreに保存されていくので　score ボタンをクリックして見ることができる
1. 繰り返し
